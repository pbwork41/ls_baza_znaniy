# Рассрочка в Сбербанке (Ю-касса)

Рассрочки Сбера оформляем через **Ю-кассу**.

[Открыть Ю-кассу →](https://yookassa.ru/my/invoice?search=496408-1678)

---

## Шаг 1. Создать счёт

Нажимаем **Создать счёт** → **Одноразовый по ссылке**

![Создать счёт](https://raw.githubusercontent.com/pbwork41/ls_baza_znaniy/main/banks/sber-images/image3.png)

---

## Шаг 2. Заполнить данные

- Указываем **ФИО клиента**
- Для удобства можно добавить комментарий (номер клиента и имя)
- Можно ограничить клиента по дате оплаты счёта
- Для удобства можно указать номер заказа, чтобы не потерять клиента
- Нажимаем **Добавить товары**

![Данные клиента](https://raw.githubusercontent.com/pbwork41/ls_baza_znaniy/main/banks/sber-images/image4.png)

---

## Шаг 3. Добавить абонемент

Указываем наименование абонемента → цену → **Создать счёт**

![Добавить абонемент](https://raw.githubusercontent.com/pbwork41/ls_baza_znaniy/main/banks/sber-images/image1.png)

---

## Шаг 4. Отправить клиенту

Нажимаем **Скопировать ссылку** и направляем клиенту.

![Скопировать ссылку](https://raw.githubusercontent.com/pbwork41/ls_baza_znaniy/main/banks/sber-images/image2.png)
